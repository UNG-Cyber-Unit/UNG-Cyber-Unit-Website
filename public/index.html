// About the Unit

The UNG Cyber Unit is a student cybersecurity group at the University of North Georgia in Dahlonega, Georgia. We help students build real security fundamentals — from the CIA triad and password hygiene to phishing defense, networking, encryption, and the Linux command line — through short, hands-on lessons and quizzes.

Whether you're brand new to security or sharpening up for a Capture the Flag (CTF) competition, the Cyber Unit's free learning modules cover the core concepts every defender and analyst should know. Instructors can also host live Quiz Rooms for classes and practice sessions.

Explore the Core Topics above, browse our security resources, or read more about the unit and its place in the University of North Georgia cyber community.
