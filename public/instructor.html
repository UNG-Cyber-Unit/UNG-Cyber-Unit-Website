📄 File Format Guide ▼

CSV Format

ColumnRequired?Notes
question requiredQuestion text
type optionalmultiple_choice (default) or free_response
answer_a required*First answer choice — *not needed for free_response
answer_b required*Second answer choice — *not needed for free_response
answer_c optionalThird choice — leave blank if unused
answer_d optionalFourth choice — leave blank if unused
correct required*0-indexed — 0=A, 1=B, 2=C, 3=D — *not needed for free_response
explanationoptionalShown to student after submit; doubles as grading notes for free_response
 
question,type,answer_a,answer_b,answer_c,answer_d,correct,explanation
"What does CIA stand for?",,"Confidentiality Integrity Availability","Central Intelligence Agency","Cyber Intrusion Analysis","Classified Information Access",0,"The CIA Triad is the foundation of cybersecurity."
"True or False: Reusing passwords is safe.",,"True","False",,,1,"Reusing passwords means one breach exposes all accounts."
"Explain why reusing passwords across sites is risky.","free_response",,,,,,"Look for: credential stuffing, one breach exposes all accounts."

Tip: build in Excel or Google Sheets, then export as CSV UTF-8.

JSON Format

An array of question objects. type is optional (multiple_choice default, or free_response). For multiple_choice, answers is a list of 2–4 strings and correct is the 0-based index of the right answer. For free_response, omit answers/correct — students type a response and you grade it manually afterward. 

[
  {
    "question": "What does CIA stand for?",
    "answers": [
      "Confidentiality Integrity Availability",
      "Central Intelligence Agency",
      "Cyber Intrusion Analysis"
    ],
    "correct": 0,
    "explanation": "The CIA Triad is the foundation of cybersecurity."
  },
  {
    "question": "True or False: Reusing passwords is safe.",
    "answers": ["True", "False"],
    "correct": 1,
    "explanation": "Reusing passwords means one breach exposes all accounts."
  },
  {
    "question": "Explain why reusing passwords across sites is risky.",
    "type": "free_response",
    "explanation": "Look for: credential stuffing, one breach exposes all accounts."
  }
]

explanation is optional. Save the file as .json.
